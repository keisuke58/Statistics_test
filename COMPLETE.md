## ⚠️ 注意事項

### Conda環境の削除

環境を削除する場合：

```bash
conda env remove -n statics_test
```

### NumPy/SciPyの互換性

Conda環境を使用することで、パッケージの互換性問題を回避できます。
pipを使用する場合で互換性エラーが発生した場合は以下を実行してください：

```bash
pip uninstall numpy scipy
pip install numpy==1.26.4 scipy==1.11.4
```
