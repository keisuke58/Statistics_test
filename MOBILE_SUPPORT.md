# 📱 モバイル対応について

## ✅ スマホから使用可能

統計検定マスターは、**スマートフォンからも使用できます**。

### アクセス方法

1. **Streamlit Cloudにデプロイ済みの場合**
   - スマホのブラウザでStreamlit CloudのURLにアクセス
   - 自動的にモバイル表示に最適化されます

2. **ローカルで実行している場合**
   - 同じWi-Fiネットワークに接続
   - スマホのブラウザで `http://[PCのIPアドレス]:8501` にアクセス

### モバイル対応機能

✅ **レスポンシブデザイン**
- 画面サイズに応じて自動的にレイアウトが調整されます
- 768px以下の画面（スマホ）で最適化された表示

✅ **タッチ操作対応**
- ボタンは最小44pxの高さでタッチしやすく設計
- 入力フィールドはiOSでズームを防ぐ設定

✅ **サイドバーの自動折りたたみ**
- モバイルではサイドバーが自動的に折りたたまれます
- ハンバーガーメニューからアクセス可能

✅ **フォントサイズの最適化**
- モバイルで読みやすいフォントサイズに調整

✅ **カラムレイアウトの調整**
- モバイルでは縦並びに自動調整

### 推奨ブラウザ

- **iOS**: Safari, Chrome
- **Android**: Chrome, Firefox
- **その他**: 最新のモダンブラウザ

### 注意事項

- グラフや表はモバイルでも表示されますが、横スクロールが必要な場合があります
- 問題文が長い場合は、スクロールして確認してください
- 統計計算ツールの入力は、モバイルでも可能ですが、PCの方が入力しやすい場合があります
